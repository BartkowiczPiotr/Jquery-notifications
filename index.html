Jquery Windows System

Clean & simple

Simple script to create windows with external content

Try it yourself

How it works?

  1. Bind click action to html element (ex: button with "show-default-window" id)
  2. Create folder and set a path (or leave it) then create there a file with content
  3. Create new SystemWindows object and set couple options:
    • window_identificator it's a name of the window and name of the file with content what you want to display in window
    • path -you can set a path to the file with external content
    • extension -you can change extension of file, default is html
    • window_title it's a title which will be displayed in header of window
    • width this is width of window, expressed in percents. Default value is 40 (40%)
  4. Call createWindow() method
  5. Done!
    6. To work properly, script is using jquery-ui.js. You have to use it on real server or on localhost
